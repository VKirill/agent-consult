# Security Policy

## Supported Versions

Only the latest release version of **Agent Consult** is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you find any vulnerabilities (e.g. key leaks, credential leakage through sandbox isolation, arbitrary code execution), please do **NOT** open a public GitHub issue. 

Instead, report it privately to the project maintainer:
* **Telegram**: Send a message to [t.me/pomogay_marketing](https://t.me/pomogay_marketing) or contact the owner directly.
* **GitHub Profile**: [VKirill](https://github.com/VKirill)

Please include:
1. A detailed description of the vulnerability.
2. Steps to reproduce or a Proof of Concept (PoC).
3. The potential impact of the vulnerability.

We will review your report within 48 hours and work with you to release a security patch as soon as possible. Thank you for helping keep this project secure!
